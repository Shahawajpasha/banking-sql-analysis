# Banking Transaction SQL Analysis

## Overview
This project analyzes synthetic banking transaction data using SQL to generate business insights, customer behavior patterns, and risk indicators.

## Key Questions Answered
- Monthly revenue trends
- Top customers by revenue
- Category-wise spending
- Weekday vs weekend behavior
- High-risk and suspicious transactions
- Customer lifetime value

## Tools Used
- SQLite / SQL
- CSV for outputs
- GitHub for version control

## Files
- analysis.sql — All analysis queries
- *.csv — Query outputs

## Insights
- Identified high-value customers driving most revenue
- Found higher transaction volume during weekdays
- Detected suspicious high-value transactions for risk monitoring
